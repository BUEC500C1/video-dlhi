## FFMpeg setup
Programmed on Windows Linux Subsystem running Ubuntu 18.04.4 LTS

1. Download ffmpeg with
```
    sudo apt-get install ffmpeg
```
2. Download ffmpeg python repository with
```
    pip3 install ffmpeg-python
```
